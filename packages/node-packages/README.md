# NodeJS Packages

Any NPM packages not available under regular nixpkgs can be added here.

## Structure

Most files are generated by node2nix.
`node-packages.json` specifies packages which must be installed.
`override.nix` patches some packages to make them install correctly.

## Building/Updating

```bash
nix-env -f '<nixpkgs>' -iA nodePackages.node2nix
node2nix -i node-packages.json
```

## Testing

```bash
nix-buld -A react-site override.nix
```

## Installing Locally

```bash
nix-env -f override.nix -iA gatsby-cli
```
